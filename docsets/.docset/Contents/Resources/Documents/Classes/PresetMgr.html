SoundBank specific methods
  • getMelodicInstrumentFromSoundBank(id:path:isSoundFont:)

    Builds an AUSamplerInstrumentData structure corresponding to the Sound Bank file at the provided path. This method is specific to melodic instruments; If you want to load a percussion soundbank see getPercussionInstrumentFromSoundBank. A basic check is done to assertain wether the file has the correct format. Beware this is based only on the file extension!

    Declaration

    Swift

    func getMelodicInstrumentFromSoundBank(id: UInt8 = 0, path: String, isSoundFont: Bool = true) -> AUSamplerInstrumentData?

    Parameters

    id

    The patch’s id in the given sound bank file
    path

    The path to the sound bank file.
    isSoundFont

    Determines if the file should be treated as a SoundFont file (e.g. *.sf2) or not (e.g. *.dls)

    Return Value

    • If `path` is valid: The corresponding structure
    • Else: returns `nil`.
